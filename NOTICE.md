# Notices

This standalone repository contains the `thesis-idea` Codex skill, publicly named:

- 中文名：毕业论文 idea 打磨器
- English name: Thesis Idea Builder

Original content in this repository is licensed under CC BY-NC 4.0 unless otherwise stated.

The IRIS-style idea tree is inspired at the abstract method level by the IRIS paper:

Garikaparthi, Aniketh, Manasi Patwardhan, Lovekesh Vig, and Arman Cohan. 2025. "IRIS: Interactive Research Ideation System for Accelerating Scientific Discovery." ACL 2025 System Demonstrations. DOI: 10.18653/v1/2025.acl-demo.57.

This repository does not include IRIS source paths, source mappings, source code, prompts, UI, README wording, or setup instructions.

This repository intentionally does not include `econ-write`, `cn-top-econ-writing`, `econ-table-figure-design`, or other skills from the broader economics writing workflow repository.
